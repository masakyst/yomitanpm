Yomitan Perl Mongers ( Yomitan.pm
====================

沖縄県(中部)を中心とするPerl ユーザのコミュニティ形成を目指す非営利の団体です。 主な活動内容はプログラミング言語 Perl に関係するメンバー主催の勉強会やインターネット上での啓蒙活動や情報交換です。 どなたでも無料で参加できます。連絡先/情報発信は、 twitter @yomitanpm です。
